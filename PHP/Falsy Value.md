# Falsy Value

It is also apply to `boolval()`.

- (boolean) FALSE
- (integer) 0
- (float) 0.0, -0.0
- (string) "0"
- (string) ""
- (array) no element
- (null) NULL (including unset variables)
- (SimpleXML) attributeless empty elements which have neither children nor attributes.

Every other value is considered TRUE.
